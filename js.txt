Alur
User
1. User Login
2. User memilih barang di dashboard
3. User menambah barang di katalog ke keranjang
4. User melakukan proses pesanan dengan mengisi form sbb :
    1. jumlah barang;
    2. alasan. -> parameter
		a. habis
		b. hilang
		c. rusak
		d. stok
    3. dokumentasi jika (rusak)
5. User akan mencek ulang list pesanan dan lalu mengisi nomor surat pesanan
6. Op Login
7. Op membukan menu transaksi
8. Op memproses pesanan
9. Op mensortir jumlah pesanan yg dapat diminta
10. Jika ada barang yg termasuk barang yg perlu aproval maka OP menunggu aproval dari Pimp
11. Pimp Login
12. Pimp membuka menu aproval
13. Pimp melakukan Aproval pada permintaan user
14. Op menyiapkan barang yang sudah disortir dan Aproval (Jika ada)
15. Op merubah status pesanan pada aplikasi
16. Op menyerahkan barang pada user 
17. User menandatangani surat
